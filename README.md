# Polynomial

C++ проект, объединяющий библиотеку для работы с алгебраическими многочленами (полиномами) и модуль численного интегрирования математических функций.

## 🧮 Описание модулей

### 1. Библиотека многочленов (`Polynomial.h` / `Polynomial.cpp`)
Класс `Polynomial` реализует полноценную алгебру многочленов произвольного порядка:
- **Арифметические операции**: Сложение, вычитание, умножение, деление и взятие остатка от деления многочленов.
- **Математические функции**:
  - Нахождение производной многочлена.
  - Нахождение наибольшего общего делителя (НОД) двух многочленов (алгоритм Евклида для полиномов).
  - Устранение кратных корней многочлена.
- **Ввод/вывод**: Перегрузка операторов потока `<<` и `>>` для естественного ввода и вывода (например, `x^2 - 4x + 4`).

### 2. Численное интегрирование (`myFunctions.h` / `main.cpp`)
Модуль для вычисления определенных интегралов вещественных функций. В демонстрационной программе производится расчет интеграла для функции $F(x) = \frac{1}{\ln(x)}$ на интервале от $2$ до $5$:
- **Метод прямоугольников** (`rectangleMethod`).
- **Метод трапеций** (`trapezoidMethod`).
- **Метод Симпсона (парабол)** (`simpsonsMethod`).

## 📦 Сборка и запуск

Проект поставляется с конфигурацией сборки CMake. Для сборки требуется компилятор (GCC/Clang/MSVC) и CMake.

### Шаги для сборки:
```bash
# Создайте директорию сборки
mkdir build
cd build

# Сгенерируйте сборочные файлы и скомпилируйте проект
cmake ..
cmake --build .
```

### Запуск исполняемого файла:
```bash
# Запуск из директории сборки
./Polynomial
```

При запуске программа выполнит численное интегрирование функции $f(x) = \frac{1}{\ln(x)}$ с разбиением на 100 000 интервалов и выведет результаты трех различных методов:
```text
F(x) = 1 / log(x)
Интеграл от 2 до 5
Метод прямоугольников: 2.44686
Метод трапеций: 2.44686
Метод Симпсона: 2.44686
```
